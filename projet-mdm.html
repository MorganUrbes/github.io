Simplifier la gestion de milliers de terminaux mobiles

RÔLEDesigner UX UI
ÉQUIPE9 personnes
DURÉE6 mois
ANNÉE2021

Le contexte

Mobile Device Management est un outil permettant aux administrateurs informatiques de gérer plusieurs milliers de terminaux mobiles répartis dans différents pays.

La mission

Concevoir une expérience plus simple pour des utilisateurs aux rôles très différents tout en assurant la cohérence du produit.

Focus sur le projet.

Comprendre les usages avant de concevoir

Les enjeux

Des utilisateurs aux profils variés avec des besoins et des niveaux d'expertise très différents.

Adapter les interfaces selon le rôle

Tous les utilisateurs n'ont pas les mêmes responsabilités. Les parcours doivent être personnalisés selon les permissions.

Réduire la charge cognitive

Les informations essentielles doivent être priorisées afin de faciliter les actions les plus fréquentes.

Uniformiser les composants

Les nouveaux écrans doivent s'appuyer sur le Design System afin d'améliorer la cohérence du produit.
