Documentation API

Swisslife
Plateforme de documentation API

Concevoir une plateforme facilitant la compréhension et l'intégration des API

RÔLEProduct Designer
ÉQUIPE2 personnes
DURÉE2 mois
ANNÉE2023

Le contexte

Dans un contexte d'ouverture des services, Swiss Life souhaitait proposer une plateforme de documentation moderne facilitant l'intégration de ses API. L'objectif était de rendre des contenus techniques accessibles et compréhensibles pour les différents profils concernés.

La mission

Participer à la conception de la plateforme, depuis les premières phases d'exploration jusqu'à la validation fonctionnelle après développement.

Focus sur le projet.

Rendre le technique accessible et compréhensible

Les enjeux

Concevoir une plateforme technique qui soit compréhensible et utilisable par des profils très différents.

Comprendre les profils utilisateurs

Les besoins varient selon qu'on est développeur, partenaire ou décideur.

Rendre le contenu accessible

Transformer une documentation technique dense en contenu clair et facile à naviguer.

Valider avant de livrer

S'assurer que la plateforme répond aux attentes réelles avant la mise en production.
