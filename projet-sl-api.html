Documentation API

Swisslife
Plateforme de documentation API

Concevoir une plateforme facilitant la compréhension et l'intégration des API

RÔLEProduct Designer
ÉQUIPE2 personnes
DURÉE2 mois
ANNÉE2023

Le contexte

Dans un contexte d'ouverture des services et de multiplication des échanges entre systèmes, Swiss Life souhaitait proposer une plateforme de documentation moderne facilitant l'intégration de ses API. L'objectif était de rendre des contenus techniques accessibles, compréhensibles et simples à utiliser pour les différents profils concernés.

La mission

Participer à la conception de la plateforme, depuis les premières phases d'exploration jusqu'à la validation fonctionnelle après développement.

Focus sur le projet.

Rendre la documentation technique accessible à tous

Les enjeux

L'enjeu était de transformer une documentation technique en une expérience de consultation claire et efficace. Chaque décision de conception devait faciliter l'accès à l'information, accompagner les développeurs dans l'intégration des API et sécuriser la qualité du produit avant sa mise en production.

Rendre une documentation technique accessible

Faciliter la compréhension des API grâce à une interface claire et une organisation intuitive des contenus.

Accompagner les développeurs

Permettre une prise en main rapide des API grâce à une navigation simple et des exemples facilement identifiables.

Garantir la qualité avant mise en production

Vérifier que les développements respectaient les maquettes et répondaient aux besoins exprimés.
